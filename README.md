# Metanoia-LAB
Ecossitema de gestão de conhecimento Biblico, filosófico, histórico, apologético

o sistema deseja atender:
 - estruturação e consumo de conteúdos intelectuais. 
 - compartilhamento facilitado de conhecimento
 - acessibilidade multi-formato: (áudio, texto, slides, resumo por IA)

Stack Tecnológica prevista:
 - Backend: FastAPI (python)
 - Banco de dados: Postgres
 - Frontend: Next.js + TailwindCSS + ShadcnUI
 - Processamento: Langchain
 - Infra: servidor + nginx e Docker
